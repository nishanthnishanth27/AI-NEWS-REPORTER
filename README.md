🚀 Introduction
The AI News Reporter is an advanced web application that leverages modern AI technologies to fetch the latest news and provide instant summaries in Tamil. It acts as a digital news anchor, specifically designed to help Tamil-speaking users stay updated with global and technical insights quickly and efficiently.

💡 Key Features
1. AI-Powered Tamil Summarization
The app utilizes the Gemini 1.5 Flash API to analyze complex news articles and generate concise, two-line summaries in Tamil. This allows users to grasp the core of any story instantly.
2. Real-Time Activity Tracking (EmailJS)
To enhance engagement and security, the system is integrated with EmailJS. Whenever a user opens the app or performs a search, an automated notification is sent to the developer's personal email. This feature is a "mass logic" highlight for the technical symposium.
3. SQL Database Integration
Users can bookmark important news articles using the "Save SQL" feature. These records are stored in a local SQLite database via a Python Flask backend, demonstrating full-stack capabilities during live presentations.
4. Interactive Voice Search
The application includes a Voice Search feature, allowing users to find news using voice commands in English or Tamil, making the interface highly accessible.
5. Seamless User Experience
A custom Splash Screen loading logic ensures that the AI engine initializes smoothly before the user enters the main dashboard, providing a professional and polished feel.
🛠️ Tech Stack
Frontend: HTML5, CSS3 (Tailwind CSS Framework), JavaScript
AI Engine: Google Gemini 1.5 Flash API
Backend: Python Flask API
Database: SQLite (for local news archiving)
Email System: EmailJS (for real-time search and login alerts)
